# Inquira

Live Demo: https://inquira.freehosting.dev/

GitHub Repository: https://github.com/ShreyaaNchavan/Inquira

 About Inquira
---

Inquira is a modern Question & Answer web application developed using PHP, MySQL, Bootstrap, HTML, CSS, JavaScript, and AJAX. It provides a platform where users can ask questions, answer others' queries, browse by categories, search discussions, and interact through likes.

The name "Inquira" is inspired by the word "Inquire", which means to ask or seek information. It represents a community-driven platform where users learn by asking questions and sharing knowledge.

---

##  Features

### 1. User Authentication
- User Registration
- Secure Login & Logout
- Password Hashing using PHP
- Session Management

### 2. Question Management
- Ask Questions
- View All Questions
- View Latest Questions
- Search Questions
- Filter Questions by Category
- View My Questions
- Delete Your Own Questions

### 3. Answer System
- Answer Questions
- View All Answers
- Related Questions Section
- Only Logged-in Users Can Answer

### 4. Like System
- Like Answers
- AJAX-based Like Feature (No Page Refresh)

### 5. Categories
- Browse Questions by Category
- Dynamic Category List

##  Installation

### 1. Clone the Repository

```bash
git clone https://github.com/ShreyaaNchavan/Inquira.git
```

### 2. Move Project

Copy the project folder into

```
xampp/htdocs/
```

### 3. Create Database

Open

```
http://localhost/phpmyadmin
```

Create a database named

```
discuss
```

### 4. Import Database

Import

```
discuss.sql
```

into the database.

### 5. Configure Database

Update

```
common/db.php
```

with your database credentials.

### 6. Start XAMPP

Start

- Apache
- MySQL

### 7. Run Project

Visit

```
http://localhost/discuss
```
